# 数组
::: warning
 - 数组长度是固定的，不能动态修改，且必须在定义数组时指定长度
 - 数组元素必须是相同的数据类型
:::

## 数组定义
::: tip 默认值
  1. 整数类型默认值是0
  2. 浮点数类型默认是0.0
  3. 字符类型默认是空字符
  5. 引用类型默认是null
  6. 布尔类型默认是false
:::
```java
  int[] arr = new int[3]; 
  arr[0] = 100;
  int[] arr2 = {1,2,4,5};
  String[] strArr = {"html","css","java"};
```

## 数组遍历
```java
public static void show() {
    int[] arr = new int[3];
    arr[0] = 100;
    arr[1] = 200;
    arr[2] = 300;
    for(int item:arr) {
        System.out.println(item);
    }
    for(int i=0;i<arr.length;i++) {
        System.out.println(arr[i]);
    }
}
```
