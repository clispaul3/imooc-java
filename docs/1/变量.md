# 变量

## 标识符
  - 标识符就是用于给Java程序中变量、类、方法、包名等命名的符号
  - 标识符就是变量名、类名、方法名等

## 描述变量
  1. 变量类型
  2. 变量名
  3. 变量值

## 命名规范
  - 只能包含数字、字母或下划线
  - 不能以数字开头
  - 不能有空格
  - 区分大小写
  - 见名知意

## 使用变量
  - 声明变量
  ::: warning
   - 声明变量，必须指定变量的数据类型
   - 变量在使用前，必须先初始化
   - 变量声明时，可以不初始化
  :::
  - 变量赋值
  - 变量修改
  ::: warning
