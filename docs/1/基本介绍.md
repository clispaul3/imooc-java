## 使用Vue组件
  `.vuepress / components 中的任意 * .vue 文件都会自动注册为全局的异步组件`
  ```js
  // 定义组件
  .
  └─ .vuepress
      └─ components
        ├─ demo-1.vue
        └─ OtherComponent.vue

  // 在markdown文件中使用组件
  <demo-1/>
  <OtherComponent/>
  ```
